# FNANCZAE1_Script_Builder

A Script that builds Five Night's at NCZ game made by MM33 to a usable linux executable

*Basically a FNAF fan made game.*

> [!NOTICE]
> This script is still experimental so if you experience bugs, let me know in discord

# Linux Installation Guide

*< If there were errors during the installation, let me know on discord. >*

**FOR VERSIONS 1.3.0+**
- Download the game zip from discord
*Note: make sure you have `unzip` and `wget` packages installed beforehand*
- Run this command line
```
rm -f fnancz-installer2.sh >/dev/null 2>&1 && wget https://raw.githubusercontent.com/AmrThePigeon/FNANCZAE1_Script_Builder/refs/heads/main/fnancz-installer2.sh --no-cache && chmod +x fnancz-installer2.sh && ./fnancz-installer2.sh
```
Enjoy playing the game!

**FOR VERSIONS 1.2.0 AND OLDER**
- Download the game zip from discord
*Note: make sure you have `unzip` and `wget` packages installed beforehand*
- Run this command line
```
rm -f fnancz-installer1.sh >/dev/null 2>&1 && wget https://raw.githubusercontent.com/AmrThePigeon/FNANCZAE1_Script_Builder/refs/heads/main/fnancz-installer1.sh --no-cache && chmod +x fnancz-installer1.sh && ./fnancz-installer1.sh
```
