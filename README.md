# FNANCZAE1_Script_Builder
A Script that builds Five night's at NCZ game made by MM33 to a usable linux executable

# Linux Installation Guide
- Download the game zip from discord
*Note: make sure you have `unzip`, `wget` and `curl` packages installed beforehand*
- Run this command line
```
curl -s 
```
Good luck, the game is installed
(and don't mind the wget and curl in same needs because I couldn't test it due to some problems with my packages)
